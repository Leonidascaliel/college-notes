### LAN vs WAN
* LAN: Local Area Network
Exemplo de LAN: Rede de LanHouse, conexão entre os computadores para jogar.
* WAN: Wide Area Network
1. Interconecta LANs em áreas amplas (Cidades, estados, países);
2. Administrada por provedores de serviços (ISPs);
3. Geralmente fornece links de velocidade mais lenta entre LANs do que as conexões internas de LAN.

### A Internet
* Não pertence a ninguém
É definida como uma infraestrutura global de redes de computadores interconectadas que utilizam o conjunto de protocolos **TCP/IP** (Transmission Control Protocol/Internet Protocol) para se comunicar e trocar dados. É uma "rede de redes" que conecta milhões de dispositivos, incluindo servidores, computadores e smartphones, através de meios físicos, sem fio e ópticos. 

Aqui está uma análise técnica detalhada da internet:

1. Funcionamento: Comutação de Pacotes

A internet não envia dados em um fluxo contínuo. Ela utiliza a **comutação de pacotes**. 

- **Dados:** Arquivos, imagens e vídeos são quebrados em pedaços menores, chamados de pacotes.
- **Encaminhamento:** Cada pacote é enviado individualmente, podendo percorrer rotas diferentes pela rede para evitar congestionamentos.
- **Remontagem:** No destino, os pacotes são reordenados e montados novamente. 

2. Protocolos Base (TCP/IP)

A comunicação técnica é regida por protocolos padronizados: 

- **IP (Internet Protocol):** Responsável pelo endereçamento, garantindo que os dados cheguem ao local correto. Cada dispositivo tem um IP (IPv4 ou IPv6).
- **TCP (Transmission Control Protocol):** Responsável por garantir que a entrega dos pacotes seja confiável, verificando erros e reordenando-os. 

3. Infraestrutura Física e Lógica

A "rede de redes" é sustentada por:

- **Backbone:** Cabos de fibra óptica de alta velocidade e satélites que formam a espinha dorsal da internet.
- **Roteadores:** Equipamentos que direcionam o tráfego de dados entre redes diferentes, definindo a melhor rota para os pacotes.
- **DNS (Domain Name System):** O "catálogo de endereços" da internet que converte nomes de domínio (ex: [google.com](https://www.google.com/url?sa=i&source=web&rct=j&url=https://www.google.com/&ved=2ahUKEwi1joqjyNaSAxVLD7kGHVyeKugQy_kOegQIDBAD&opi=89978449&cd&psig=AOvVaw3nl53MXWX4K3WPyIxKd8ib&ust=1771075435882000)) em endereços IP numéricos.
- **CDN (Content Delivery Networks):** Redes de servidores distribuídos geograficamente para armazenar conteúdos (como vídeos) mais perto do usuário e aumentar a velocidade. 

4. Camadas da Internet (Modelo OSI/TCP-IP) 

A comunicação é organizada em camadas, cada uma com uma função específica (do físico ao usuário): 

1. **Física:** Cabos de fibra óptica, cabos metálicos, ondas de rádio (Wi-Fi), responsáveis pela transmissão de bits brutos.
2. **Enlace (Data Link):** Gerencia a conexão física, detecção e correção de erros no meio físico.
3. **Rede (IP):** Roteamento e endereçamento de pacotes através de diferentes redes.
4. **Transporte (TCP/UDP):** Controle de fluxo e garantia de entrega entre origem e destino.
5. **Aplicação:** Camada superior onde operam protocolos como HTTP/HTTPS (navegação), SMTP (email) e FTP (transferência de arquivos). 

6. Estrutura de Conexão

- **LAN/WAN:** Dispositivos residenciais/corporativos (LAN) conectam-se a provedores de serviços (ISP) via WAN.
- **Pontos de Troca de Tráfego (IX.br/NAPs):** Onde os provedores de internet se interconectam para trocar tráfego.

#### TTL ou "Time to Live"
* define a duração ou o número de "saltos" (hops) que um pacote de dados pode percorrer numa rede antes de ser descartado por um roteador
