### O Que é uma Classe?
Uma **Classe** representa um conjunto de **elementos** ([[Objetos em POO]]) que possuem as mesmas **propriedades** (Atributos) e as mesmas **operações** (Métodos).

Exemplo de Classe (Conjunto):
* Aeronaves = Helicóptero, avião...
* Mamífero = Cavalo, Urso, Vaca...
### Atributos de uma Classe
Os atributos de uma classe representa uma estrutura de dados e os métodos que representam um código definido a partir de uma lógica de programação.
#### A representação de uma classe é feita a partir da seguinte simbologia:
1. Nome da classe
2. Atributos: Tipo do dado = Valor inicial
3. Operação (Lista de segmentos): Tipo do resultado

### Definição de Atributos
Todos os objetos de uma classe terão os mesmos **atributos**. Cada atributo representam um tipo de dado (Inteiro, real, caractere, matriz, ponteiro, etc...). A diferença entre os objetos de uma mesma classe é representada pelo armazenamento de diferentes valores para estes seus atributos.

### Definição dos Métodos
Todo método é um bloco de instruções cuja finalidade é permitir que o objetivo execute uma determinada operação pré-estabelecida. Os objetos instanciados a partir de uma mesma classe utilizam os mesmos métodos (Código) e somente os métodos do próprio objeto terão acesso sobre os seus atributos.

### Exemplo de Criação de Classe
Objeto ([[Objetos em POO]]) = Caneta

1. nome = Caneta;
2. Atributos = Tamanho, tipo de escrita, cor, constituição;
3. Operação = Escrever();

A partir desse objeto, é possível criar um outro objeto semelhante, com os mesmos atributos (Características) e a mesma operação, entretanto, com valores de atributos diferentes.
