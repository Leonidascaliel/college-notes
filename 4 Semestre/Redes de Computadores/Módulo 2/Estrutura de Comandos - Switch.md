Para visualizar os comandos no console do Switch precisa acessar um terminal de uma máquina no Cisco Packet Tracer ou no computador com uma conexão serial com o Switch.
Após acessar o Terminal, você pode usar o comando "?" para visualizar a lista de comandos disponíveis do modo. Caso não tenha o comando que procura, utilize de documentação ou IA para facilitar o processo.

* Promtp: Indica omodo
* Comando: a ação (description)
* Argumento: O valor (Servidor_RH)]
